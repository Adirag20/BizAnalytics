# BizAnalytica

A real-world business analytics tool that analyzes sales data to extract meaningful insights.

## Features
- Generates realistic dummy sales data.
- Calculates total revenue and top-selling products.
- Visualizes monthly revenue trends.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Generate data:
   ```bash
   python data_generator.py
   ```
2. Run analysis:
   ```bash
   python analysis.py
   ```
